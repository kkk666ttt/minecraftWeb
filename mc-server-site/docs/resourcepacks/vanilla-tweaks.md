# Vanilla Tweaks 资源包

## 简介

Vanilla Tweaks 是一个在保留 Minecraft 原版风格基础上进行细节优化的资源包。它不会改变游戏的核心视觉风格，而是通过小修小补让游戏体验更加舒适。

## 功能特性

- **自定义 3D 模型**：工具、武器、食物等物品拥有更精致的 3D 模型
- **视觉优化**：改进的光影效果、更清晰的 GUI 界面
- **GUI 美化**：菜单界面、容器界面等经过重新设计
- **连接纹理**：玻璃、书架等方块拥有无缝连接纹理

## 安装方法

1. 下载资源包文件（.zip 或 .mcpack）
2. 将文件放入游戏目录的 `resourcepacks` 文件夹
3. 在游戏设置中启用资源包
4. 如有需要，调整资源包加载顺序

## 兼容性

- 支持 Minecraft 1.21.4 及以上版本
- 兼容 OptiFine、Sodium、Iris 等主流模组
- 可与光影包同时使用

## 注意事项

- 某些 3D 模型可能需要 OptiFine 或 CIT 支持
- 如果遇到纹理显示异常，请尝试重新加载资源包
- 建议与其他材质包搭配时注意加载顺序
