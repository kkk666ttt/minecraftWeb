# LightUP 照明插件

## 概述

LightUP 是一个提供各种照明效果和功能的插件，让你轻松点亮你的建筑和基地。

## 安装要求

需要安装 CommandAPI 11.2.0 作为前置依赖。

- 下载链接：[CommandAPI-11.2.0-Paper.jar](https://cdn.modrinth.com/data/ExxvCi0y/versions/wtbh8UEY/CommandAPI-11.2.0-Paper.jar)
- 官方页面：[https://hangar.papermc.io/LOOHP/LightUp](https://hangar.papermc.io/LOOHP/LightUp)

## 指令列表

| 指令 | 说明 |
|------|------|
| `/lightup` | 基础点亮功能 |
| `/lightup undo` | 撤销上一次操作 |
| `/lightup cancel` | 停止当前点亮进程 |
| `/lightup reload` | 重载插件配置（管理员专用） |

## 使用示例

1. 手持你要放置光源的物品
2. 输入 `/lightup` 开始点亮模式
3. 右键点击你要放置光源的位置
4. 输入 `/lightup undo` 可以撤销上一步
5. 完成操作后输入 `/lightup cancel` 退出模式

## 注意事项

- 请确保你有足够的建筑权限
- 光源放置后可以通过常规方式破坏
- 管理员可以通过 `/lightup reload` 重载配置
